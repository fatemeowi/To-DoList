# Cute Digital Planner

A simple and aesthetic planner application built with HTML, CSS, and JavaScript.

## Features

* Today task management
* Tomorrow planning section
* Important notes area
* Task progress tracker
* Dynamic task creation
* Task completion tracking
* Clean glassmorphism-inspired design

## Built With

* HTML5
* CSS3
* JavaScript

## Project Structure

* **Today** → Manage daily tasks
* **Tomorrow** → Plan upcoming tasks
* **Tracker** → Monitor task completion progress
* **Important Notes** → Store quick notes and reminders

## Future Improvements

* Local Storage support
* Dark mode
* Weekly planner
* Custom themes
* Drag & drop task management
* Mobile optimization
